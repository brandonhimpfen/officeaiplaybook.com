Safe summary prompt 
Summarize this document safely.

Rules:
- Do not include identifiers, names, or sensitive details.
- If something is uncertain, say so.
- Output should be useful for a coworker who hasn't read it.

Provide:
1) 5-line overview
2) Key points (facts only)
3) Decisions / requirements (if present)
4) Risks / open questions
5) What I should verify
