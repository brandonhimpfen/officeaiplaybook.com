It’s a fast way to impose structure. ## Prompt Paste your rough notes (sanitized if needed) and request an outline plus questions to confirm.
Outline prompt 
Create a report outline from these notes.

Rules:
- Do not invent facts.
- Keep headings descriptive.
- End with “Questions to confirm” if information is missing.

Output:
- Title
- Executive summary (3–5 bullets)
- Background
- Findings (grouped)
- Options (if applicable)
- Recommendation (if applicable)
- Appendix: source notes

[PASTE HERE]
