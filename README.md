
## Where to edit

- Homepage: `index.html`
- Library index: `library/index.html`
- Item pages: `library/<slug>/index.html`
- Layouts: `_layouts/default.html`, `_layouts/item.html`
- Styles: `assets/css/style.css`
- Scripts: `assets/js/main.js`
